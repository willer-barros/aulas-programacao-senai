• Objetivo: Estruturar um texto longo de forma legível e organizada.
  • HTML: Utilizar as tags header, main, article e footer. Deve conter títulos (h1, h2) e parágrafos de texto.
  • CSS: Definir uma largura máxima (max-width) para o texto, centralizar o bloco e ajustar o line-height para melhorar a leitura.
  • Conceito chave: Tags semânticas, unidades de medida (px/rem) e espaçamento entre linhas.

Guia de Leitura: Estrutura de Texto

Exemplo de uso das tags semânticas para organizar um texto longo.

Introdução

A organização do conteúdo em tags semânticas melhora a acessibilidade e a compreensão do documento. Usar header, main, article e footer ajuda leitores e motores de busca a identificar seções importantes do texto.

Desenvolvimento

Defina uma largura máxima para o bloco de texto (max-width) para evitar linhas muito longas, que prejudicam a leitura. Ajuste o espaçamento entre linhas (line-height) para criar um fluxo confortável entre os parágrafos.

Use unidades relativas (rem) para tipografia e espaçamento para manter consistência em diferentes tamanhos de tela. Além disso, mantenha uma hierarquia clara de títulos (h1, h2, h3).

Conclusão

Aplicando essas práticas, o conteúdo se torna mais legível e agradável. A responsividade garante que o texto se adapte a dispositivos móveis sem perder proporção ou espaçamento.
