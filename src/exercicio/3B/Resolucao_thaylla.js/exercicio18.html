O Poder do HTML Semântico

O desenvolvimento web moderno vai muito além de apenas fazer um site ficar bonito visualmente. Quando estruturamos nosso código utilizando tags semânticas, estamos construindo uma base sólida para que robôs de busca, como o Google, e leitores de tela entendam perfeitamente a hierarquia da informação.

Por que a largura máxima (max-width) importa?

Estudos de legibilidade mostram que o cérebro humano se cansa rapidamente quando precisa ler linhas de texto longas demais, como aquelas que cruzam uma tela inteira de monitor de ponta a ponta. Ao limitar o tamanho do bloco de texto para aproximadamente 70 a 80 caracteres por linha (cerca de 700px), a leitura flui de forma muito mais natural.

O impacto do Line-Height

O espaçamento entre as linhas (line-height) funciona como o ar que o texto precisa para respirar. Um valor muito baixo deixa as letras espremidas, fazendo com que o leitor se perca ao pular de uma linha para a outra. O equilíbrio ideal gira entre 1.5 e 1.7, garantindo conforto visual mesmo em artigos longos.
